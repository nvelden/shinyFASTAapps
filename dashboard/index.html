Browser-based sequence tools

Tools to solve your >FASTA file problems

Each app runs fully in the browser with Shinylive. First visits can take up to 1-2 minutes while runtime assets load and cache.

  • Merge Combine two or more FASTA files
  • FASTA -> .CSV Convert FASTA to .CSV file
  • Filter Use properties to filter FASTA files
  • Parameters Calculate sequence parameters
